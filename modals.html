Uso de Modals

Para crear un modal, se utiliza la estructura básica de Bootstrap. El modal se compone de un contenedor principal con la clase .modal, que contiene un contenedor de diálogo con la clase .modal-dialog, y dentro de este, el contenido del modal con la clase .modal-content.

En el ejemplo proporcionado, se muestra un modal con un título, un cuerpo y un pie de página con botones de acción. El modal se activa mediante un botón que utiliza los atributos data-bs-toggle="modal" y data-bs-target="#nuevoModal" para especificar el modal que se desea mostrar.
